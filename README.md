# Data Mining Homework 3
## Dataset
[Video Game Sales](https://www.kaggle.com/gregorut/videogamesales)
## Task
该数据集包含游戏名称、类型、发行时间、发布者以及在全球各地的销售额数据。  
数据量：11列共1.66W数据。  
基于这个数据集，进行以下问题的探索：  
1、电子游戏市场分析：受欢迎的游戏、类型、发布平台、发行人等；  
2、可视化应用：如何完整清晰地展示这个销售故事。  
3、预测每年电子游戏销售额。  
## 提交的内容
* 对数据集进行处理的代码
* 数据挖掘代码
* 挖掘过程的报告：展示挖掘的过程、结果和分析
